# 🎉 Interactive Birthday Web Experience

A customizable and interactive birthday website template designed to deliver a complete storytelling experience with animations, mini-games, and personalized content.

This project was ideated and designed independently, with AI-assisted development used to accelerate implementation and enhance efficiency.

---
## 🌐 Live Demo
https://BackEndUJ.github.io/interactive-birthday-experience/

## 🚀 Features

- 🎬 Animated scrollytelling-style introduction
- 🎂 Interactive cake blowing mini-game
- 💌 Personalized message/letter section
- 📖 Memory book with page-flip interaction
- ✨ Smooth transitions and UI animations
- 🎨 Fully customizable content and layout

---

## 🛠 Tech Stack

- HTML5  
- CSS3 (Animations, Grid, Flexbox)  
- JavaScript (DOM manipulation, events, UI logic)  

---

## 🤖 Development Approach

This project follows an **AI-assisted development workflow**:
- Concept, flow, and UI/UX design were independently planned  
- AI tools were used to speed up coding and iteration  
- Final structure, customization, and logic adjustments were handled manually
